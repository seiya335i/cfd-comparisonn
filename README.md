<h2>なぜ「制度信用で良い」のか？</h2>
<p>6か月の期限で十分だから。制度信用は金利・手数料も低く、コスト面で有利。<br>
長期保有したい場合は、6か月ごとに決済し再度新規建て（ロールオーバー）を行えば良い。</p>

<h2>ロールオーバーとは？</h2>
<p>6か月の期限が来たら決済して、同じ銘柄で新たに空売りを始めることを言います。</p>
